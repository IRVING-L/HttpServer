> 首次将不成熟的httpServer代码发布在了github上。简单介绍一下我这个服务器项目代码

### 服务器项目代码介绍：
- 语言：
    - C++ 和 C。C++目前只涉及了class类的封装，其他的pthread库函数代码都是用C写的
- 项目技术：
    - 使用线程池实现服务器高频的并发请求
    - 使用有限状态机对http请求报文进行解析
    - 使用epoll多路复用技术监听客户端连接
    - *待完成：使用双链表+信号机制实现定时检测非活跃连接*
    - 整体的项目代码是模拟的Proactor模式（将异步I/O改为了非阻塞的同步I/O）
